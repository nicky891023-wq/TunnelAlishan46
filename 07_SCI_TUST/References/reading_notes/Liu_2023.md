## (a) 題目解剖【可查證——僅憑題目本身】

`Investigation of progressive failure mechanism of tunnel lining with material defects using discrete element method`

- **開頭元素**：泛用研究動名詞 "Investigation of" 起手，緊接第一個實質元素＝**機制**（progressive failure mechanism）——機制前置、非案例/現象前置。
- **有無方法詞**：**有**，"discrete element method" 以 "using ..." 後綴收尾。
- **構式拆解**：`[研究動作 Investigation of] + [機制 progressive failure mechanism] + [對象 tunnel lining] + [條件/缺陷限定 with material defects] + [方法 using DEM]`。
- **工藝觀察**：
  1. 典型「機制＋對象＋條件＋方法」四段式；"with material defects" 這個限定詞就是賣點所在（沒有它就是普通襯砌破壞模擬）。
  2. "progressive"（漸進）暗示論文交付的是**過程敘事**（裂縫萌生→擴展→貫通→失穩），不是單一狀態的安全評估——題目本身就承諾了時間軸/加載軸上的演化圖景。
  3. 方法詞放最後、用全名不用縮寫（discrete element method 而非 DEM），高分期刊常見做法。
