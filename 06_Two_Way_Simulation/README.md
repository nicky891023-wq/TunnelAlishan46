## 一句話
單向鏈漏掉「開裂→環軟化→收斂加大→更多開裂」的漸進互制；06 把耦合模量到的
**損傷場 D(θ,y)** 餵回小模殼勁度，閉合這個迴圈。**交換勁度/損傷、不交換力**
（力回饋＝同介面同時給位移與力＝過定，且舊樹 Picard 發散實證——只留對照組）。

## 兩層設計
- **L1 交錯式（主線）**：階段 k 的耦合損傷 → 階段 k+1 的小模殼勁度。無階段內迭代、
  成本 ≈2× 單向（~30–40h）、滯後一階段（物理正當：水位階段 5–30 天 >> 開裂瞬時）。
- **L2 階段內 Picard（驗證）**：只對濕峰 5–7 段做，欠鬆弛 ω=0.5、max 3 迭代、
  收斂判準 ‖Δu_cavity‖<5% 或 |ΔN_crack|/N<10%；**L1 vs L2 的差＝滯後誤差量化**
  （論文的方法論貢獻點）。

## 損傷映射（定案）
扇區 D = 斷鍵比率（24×15° × 5 y帶，拱腳不可斷帶恆 0，夾限 0.9）→
殼元素 `E = E0(1−D)^m`（m=1、樓地板 2.5GPa）→ **階段邊界單次寫入＋force.update**
（Wade secant 實戰驗證的更新模式；cycling 中改殼勁度＝T4 能量注入失穩教訓）。
