# 耦合模成果圖規格與量化標準（2026-07-07 Fable 定案 v1）

> 目的：把「斷鍵＝微損傷事件」轉成可交付的「裂縫演化」敘事，**凸顯核心物理主張：
> 水位升高期損傷率最顯著、退水後凍結**。全部量化定義集中本檔＝論文方法章的依據。

## 0. 量化標準（正式定義）

| 符號 | 定義 | 來源 |
|---|---|---|
| `N_bond_s` | 扇區 s 內**可斷**鍵總數（拱腳 1e100 帶排除） | G4 一次性普查（bond_census 探針，run 後執行） |
| `N_brk_s(k)` | 至階段 k 止扇區 s 的累計斷鍵數 | `cs_sk_cracks.txt` 座標分箱（24×15° × 5 y帶） |
| **`D_s(k)`** | **損傷密度** = N_brk_s(k)/N_bond_s，夾限 [0,1] | 基本場量 |
| `ΔN(k)` | 階段增量 = N(k)−N(k−1)；**濕/乾對比正規化＝ΔN(k)/階段天數** | 30d 乾、5d 升降、30d 濕的公平比較 |
| 底噪 | 兩層：①CONTROL-0 率外插（309/2450cyc）②**乾退各階段最小增量＝自校準地板**（主用②，①入附錄） | 淨增量 = ΔN − 地板 |
| **損傷帶（＝宏觀「裂縫」）** | 連續扇區滿足 `D_s > max(2×median(D), 0.02)` 的帶狀區 | 局部化判準——微損傷→裂縫的橋 |
| 纖維位置 | 斷鍵徑向分層：內緣 r<1.55／中層／外緣 r>1.85 | 彎曲張裂證據（外緣佔比） |
| **濕放大係數 A_wet** | `mean(ΔN/day, k=6)/mean(ΔN/day, k=2..5)` | 論文摘要級單一數字主張 |
| **凍結係數 A_frz** | `mean(ΔN/day, k=8..11)/mean(ΔN/day, k=6)` | 退水不可逆性主張 |

階段 1＝首次載入損傷（隧道服役史的壓縮），**單獨呈現、不入趨勢統計**——趨勢統計只用 k=2…11。
